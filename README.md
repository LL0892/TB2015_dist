# TB2015_dist

#### Deploiement

1. Faire un git push sur le dépot gandi
   ```
   git push gandi master
   ```
2. Deploy
   ```
   ssh XXXXX@git.YYY.gpaas.net 'deploy default.git'
   ```
   où XXX sont les identifiant (série de chifres) et YYY la localisation du serveur (dc0, dc1...)
